## Nexter
## Technologies
- HTML5
- SASS (scss)
- Desktop first
- Responsive
- Block Element Modifier (BEM)
- Grid layouts

## Requirements

- Live server

```
npm install -g live-server
```

### Commands

- Start dev server and compile sass files for development

``` 
npm run start
```

- Build css for production

``` 
npm run build:css
```
